How do I use it?

  1. Enter a nickname (under 25 characters). If you don't enter one, you'll be named "Anonymous". (Cuz anonymous is anonymous, what's the point?). Your name is saved.
  2. Optionally, enter a room ID. If you want to join someone else, their room ID will be in the bottom-right corner of the screen.
  3. Be a Peedy!

How do I markup text/what is markup?

  • Markup is formatting text. The newer updates also included colors and animations.
  • For bold text type **text**
  • For big text type ##text##
  • For italic text type ~~text~~
  • For underlined text type __text__
  • For line-through text type --text--
  • For scrolling text type %%text%%
  • For rainbow text type $r$text$r$
  • To make a new line
    do \n
  • To make your message a different color, put $$color$$ at the start of your message (for example $$red$$)

Pope (godmode)

  • /godmode [password] - Gives you owner privileges except this is for popes. Also gets saved into your cookies like /kingmode.
  • /pope - Enable the pope color.
  • /restart - Restarts the server (assumes it automaticaly restarts when it is turned off)
  • /update - Updates the settings, colors and blacklist without the server having to restart. SOME SETTINGS WON'T TAKE AFFECT UNTIL RESTARTED, SUCH AS PORT
  • /ip [user ID] - Get the targeted user's IP.
  • /tagsom [user ID] [tag] - Set the targeted user's tag.
  • /ipmute [user ID] - Leaks the targeted user's IP and turns every message they send from now on into their IP.
  • /ban [IP] [length in minutes] [reason] - Bans the target IP. If the length of the ban is zero or negative, it is permanent.
    • If you are used to Joseph Judge's banning system, keep in mind IPv4 addresses don't include the ":ffff:" prefix.
  • /unban [IP] - Unbans the target IP.
  • /bans - Get the full list of currently active (non-expired) bans.
  • /motd [message] - Sets the MotD (message of the day). If no parameter is given, the MotD is disabled.

Who are the moderators/admins?

This site has both moderators ("kings") and admins ("popes"). You can recognize them by their tags ("King" for moderators and "Owner" for admins). Crosscolors are used to impersonate them, so only check the tags.
